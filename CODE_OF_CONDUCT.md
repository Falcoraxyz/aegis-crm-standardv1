# Code of Conduct

## Our Pledge
We as members, contributors, and maintainers pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, visible or invisible disability, ethnicity, sex characteristics, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards
Examples of behavior that contributes to a positive environment include:
- Using welcoming and inclusive language
- Being respectful of differing viewpoints and experiences
- Gracefully accepting constructive criticism
- Focusing on what is best for the community
- Showing empathy towards other community members

Examples of unacceptable behavior include:
- Harassment, discrimination, or hateful conduct
- Trolling, insults, or personal attacks
- Publishing othersâ€™ private information without permission
- Any behavior inappropriate in a professional setting

## Enforcement Responsibilities
Project maintainers are responsible for clarifying standards and enforcing this Code of Conduct.

## Scope
This Code of Conduct applies within all project spaces, including GitHub issues, pull requests, discussions, and community spaces.
